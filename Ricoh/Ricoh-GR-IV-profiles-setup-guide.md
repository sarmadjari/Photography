# Ricoh GR IV User Custom Profile Configuration Guide

**U1 • Street**
**U2 • Portraits**
**U3 • Travel & Landscape**
---

## Introduction

The Ricoh GR IV features three User Custom Profile slots (U1, U2, U3) accessible via the mode dial. Each profile saves nearly all camera settings, allowing instant switching between optimized configurations for different shooting scenarios.

This guide provides menu-accurate settings for maximum image quality in three common situations: Street Photography, Portraits, and Travel/Landscape photography.

### How to Save User Mode Settings

1. Configure all settings in P, Av, Tv, or M mode as desired
2. Press **MENU** → **D (Customize Settings)** → **D1 User Mode**
3. Select **"Save User Mode Box"** → Choose Box 1-6
4. Name your profile (optional) → Press OK
5. Select **"Assign User Mode Dial"** → Assign Box to U1, U2, or U3

## U1 — Street Photography

*Optimized for: Fast, spontaneous shooting with zone focusing*

### Philosophy

Street photography demands speed and discretion. Snap Focus eliminates autofocus delay by pre-focusing at a set distance. Combined with a smaller aperture (f/5.6-f/8), everything from 1-2 meters to infinity stays acceptably sharp. Auto ISO ensures proper exposure in changing light while maintaining a fast shutter speed to freeze motion.

### Pro Tips for Street

- **Full Press Snap:** Press shutter fully (skip half-press) to instantly shoot at your preset Snap distance
- **Depth of Field:** At f/8 with 2m Snap, everything from ~1m to infinity is acceptably sharp
- **Quick Change:** Hold Macro button + turn front dial to quickly adjust Snap distance
- **B&W Option:** Set Image Control to Hi-Contrast B&W for classic street look

## U2 — Portraits

*Optimized for: Environmental portraits with subject separation and precise focus*

### Philosophy

Portrait photography benefits from a wide aperture (f/2.8-f/4) to create subject-background separation with pleasant bokeh. Pinpoint AF with Face/Eye Detection ensures sharp focus on your subject's eyes. Low ISO maintains maximum image quality for skin tones and fine details.

### Pro Tips for Portraits

- **Eye Detection:** Tap screen to select which eye/face to focus when multiple detected

- **Touch AF:** Set Touch AF to "AF Point+Focus" in D2 menu for tap-to-focus

## U3 — Travel & Landscape

*Optimized for: Maximum sharpness, dynamic range, and image quality*

### Philosophy

Landscape photography prioritizes image quality over speed. Using base ISO (100) delivers maximum dynamic range and minimum noise. The optimal aperture for the GR lens is f/5.6-f/8, avoiding diffraction softening at f/11+. Infinity focus or 5m Snap ensures distant subjects are sharp. A 2-second self-timer prevents camera shake.

### A1 Focus Settings

| Menu | Setting | Value |
|------|---------|-------|
| A1 | Focus | **Snap** (or Zone Select AF) |
| A1 | Face/Eye Detection | **Off** |
| A1 | Macro Mode | **Off** |
| A1 | Focus Limiter | **Off** |
| A1 | Snap Focus Distance | **∞ (Infinity)** or 5m |
| A1 | Full Press Snap | **Off** |
| A1 | Focus Peaking | **On** (if using MF) |

### A2 Exposure Setting

| Menu | Setting | Value |
|------|---------|-------|
| A2 | Exposure Mode | **Av (Aperture Priority)** |
| A2 | AE Metering | **Multi-segment** (or Highlight-weighted) |
| A2 | ISO Setting | **100 (Fixed)** |
| A2 | ND Filter | **Off** (or On for long exposure) |

> ➤ **Aperture (via front e-dial):** f/5.6 – f/8

### A3-A4 Shooting & Capture Settings

| Menu | Setting | Value |
|------|---------|-------|
| A3 | Drive Mode | **Single Frame Shooting** |
| A3 | Self-timer | **2sec** |
| A4 | File Format | **RAW+JPEG** |
| A4 | Aspect Ratio | **3:2** |
| A4 | JPEG Recorded Pixels | **L** |

### A5-A6 Image Processing & Assist

| Menu | Setting | Value |
|------|---------|-------|
| A5 | White Balance | **Daylight** (or Auto WB) |
| A5 | Image Control | **Standard** (or Vivid) |
| A5 | Peripheral Illumin. Corr. | **On** |
| A5 | D-Range Correction | **Highlight: On / Shadow: Medium** |
| A5 | Noise Reduction → Slow Shutter Speed NR | **Auto** |
| A5 | Noise Reduction → High-ISO Noise Reduction | **Off** (ISO 100) |
| A6 | Shake Reduction | **On** |
| A6 | Horizon Correction | **On** |

### Pro Tips for Travel & Landscape

- **Optimal Sharpness:** f/5.6-f/8 is the sweet spot; avoid f/11+ due to diffraction
- **Highlight-weighted:** Use for high-contrast scenes to protect bright areas
- **Golden Hour:** Set WB to Daylight to preserve warm tones, not Auto
- **Long Exposure:** Use ND Filter + M mode for motion blur (waterfalls, clouds)
- **Interval Composite:** For star trails, use Drive Mode → Interval Composite
